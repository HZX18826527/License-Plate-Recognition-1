# License-Plate-Recognition
  对中国车辆图片进行车牌定位分割及识别

### Usage:

  **1.** 在项目根目录创建 ***plate***(保存车牌分割图片)、***cut***（保存车牌字符分割图片）以及***model***（保存模型）文件夹。
  
  **2.** 修改 ***base_dir***
  
  **3.**     · 训练时，使***626***行 ***train_flag = 1***  
             · 测试时，使***626***行 ***train_flag = 0***
             
  **4.** 运行即可
